## 4. Проверка прав API-ключа Bybit

**Что проверить:**

1. Найти ключ `OeMc6t...` (MAINNET)
2. Убедиться:

> ⚠️ **КРИТИЧНО**: Если Withdraw включён — СРОЧНО выключи! Это значит что любой, кто узнает ключ, может вывести ВСЕ деньги.

## 5. Бэкап базы данных trades.db

**Что сделать на VPS:**

```bash
# Подключиться к VPS:
ssh root@176.32.37.184

# Добавить автоматический бэкап каждые 6 часов:
crontab -e

# Вставить строку:
0 */6 * * * cp /opt/crypto-bot/data/trades.db /opt/crypto-bot/data/trades.db.bak
```

**Что это делает:** Каждые 6 часов (в 00:00, 06:00, 12:00, 18:00) сервер автоматически копирует базу данных. Если что-то сломается — можно восстановить из копии.

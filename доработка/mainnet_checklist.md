# ✅ Чеклист перед MAINNET

## 🔐 Безопасность

- [ ] API-ключ **без прав вывода** (только Trade: Order + Position)
- [ ] Whitelist IP-адреса VPS на Bybit
- [ ] `.env` файл с правами `600` (только владелец может читать)
- [ ] Проверить что `.env` в `.gitignore`
- [ ] Убедиться что ключ в репозитории НЕ светится

## 💰 Финансы

- [ ] Определить стартовый `working_deposit` для MAINNET (рекомендация: $50–100)
- [ ] Пересчитать `position_size_pct` для реального баланса
- [ ] Иметь запас на счёте сверх `working_deposit` (на комиссии + буфер)

## 🛠 Технические

- [ ] Исправить баг `daily_stats` (не записывается)
- [ ] Исправить баг `bot_state` (не персистируется)
- [ ] Проверить `recv_window` на VPS (синхронизация часов)
- [ ] Установить NTP на VPS (`timedatectl set-ntp true`)

## 📊 Мониторинг

- [ ] Настроить watchdog-скрипт на VPS
- [ ] Проверить что Telegram-уведомления приходят

## 🧪 Тестирование

- [ ] Прогнать `python main.py check-config` с MAINNET-ключами

## 🚀 Запуск

- [ ] Переключить `.env`: `BYBIT_TESTNET=false`
- [ ] Перезапустить бота на VPS
- [ ] Первые 48 часов — мониторинг каждого ордера вручную

#### [MODIFY] [config.py](file:///c:/Users/admin/Desktop/Бот криптотрейдер/src/config.py) + [config.json](file:///c:/Users/admin/Desktop/Бот криптотрейдер/src/config.json)

Добавить в `TradingConfig`:
```python

trailing_tp_activation_pct: float = 0.5 # Активация trailing после +0.5%
trailing_tp_callback_pct: float = 0.3   # Откат от максимума для закрытия
```

#### [MODIFY] [trading/position_manager.py](file:///c:/Users/admin/Desktop/Бот криптотрейдер/src/trading/position_manager.py)

Добавить:
- `_trailing_high: float` — максимум цены после активации trailing
- `should_trailing_tp(price)` — проверка trailing условия
- Логика: если `price >= avg + activation_pct%` → начинаем отслеживать; если `price <= trailing_high * (1 - callback_pct%)` → закрываем

#### [MODIFY] [trading/risk_manager.py](file:///c:/Users/admin/Desktop/Бот криптотрейдер/src/trading/risk_manager.py)

В `calculate_entry_size()` — модифицировать формулу:
```python
# Базовый размер
base_qty = (entry_usd * leverage) / current_price
```

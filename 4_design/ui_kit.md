## 4. Компоненты (веб-дашборд v2.0)

### 4.1 Карточки метрик

```
┌─────────────────────┐
│  💼 БАЛАНС          │ ← Подпись (Text Secondary, 12px)
│                     │
│  $5,349.20          │ ← Значение (JetBrains Mono, 32px, White)
│  +$119.20 (+2.3%)   │ ← Изменение (14px, Profit Green)
└─────────────────────┘
   ↑ Dark Blue фон, 12px border-radius, 1px Border Gray
```

### 4.2 Таблица сделок

```
┌────┬───────────┬─────────┬──────────┬──────────┬───────────┐
│ #  │ Дата      │ Входов  │ Время    │ PnL      │ Статус    │
├────┼───────────┼─────────┼──────────┼──────────┼───────────┤
│ 47 │ 04.03     │ 2       │ 4ч 23мин │ +$46.40  │ ✅ Закрыта│
│ 46 │ 04.03     │ 1       │ 2ч 11мин │ +$38.20  │ ✅ Закрыта│
│ 45 │ 04.03     │ 3       │ 6ч 45мин │ +$34.60  │ ✅ Закрыта│
│ 44 │ 03.03     │ 2       │ 3ч 18мин │ +$41.10  │ ✅ Закрыта│
└────┴───────────┴─────────┴──────────┴──────────┴───────────┘
  ↑ Строки чередуются: Dark Navy / Dark Blue
  ↑ PnL зелёный (прибыль) / красный (убыток)
  ↑ Ховер: Card Hover градиент
```

## 6. Анимации и переходы

| Элемент | Анимация | Длительность |
|---|---|---|
| Карточки (ховер) | Подъём + тень | 200ms ease |
| Числа (обновление) | Плавный переход цвета | 300ms |
| Статус-бейдж | Пульсация (при активном) | 2s infinite |
| Графики | Плавная перерисовка | 500ms ease-out |
| Уведомления (toast) | Slide-in справа | 300ms ease |
| Таблица (ховер строки) | Подсветка фона | 150ms |

# 🎯 MVP — Минимальный жизнеспособный продукт

## Цель MVP
Запустить рабочего бота, который автоматически торгует ETH/USDT на Bybit по лонговой стратегии с усреднением и отправляет уведомления в Телеграм.

---

## Состав MVP

### ✅ Входит в MVP

| # | Компонент | Описание |
|---|---|---|
| 1 | **Bybit API клиент** | Подключение REST + WebSocket, авторизация, размещение ордеров |
| 2 | **Торговый движок** | Лонговая стратегия, усреднение (5 входов), тейк-профит ~1% |
| 3 | **Риск-менеджмент** | Контроль объёма позиций ≤ баланс, проверка ликвидации, запрет шорта |
| 4 | **Телеграм-бот** | Уведомления: вход, выход, ошибки |
| 5 | **Логирование** | Запись всех событий и сделок в файл |
| 6 | **Конфигурация** | `.env` (секреты) + `config.json` (параметры стратегии) |
| 7 | **CLI** | Запуск одной командой, graceful shutdown |
| 8 | **Бэктестинг** | Симулятор DCA-стратегии, загрузка данных, оптимизатор |

### ❌ Не входит в MVP (будущие версии)

| # | Компонент | Версия |
|---|---|---|
| 1 | Веб-дашборд | v2.0 |
| 2 | Мультивалютная торговля | v2.0 |
| 3 | Поддержка других бирж (Binance, OKX) | v3.0 |
| 4 | Мобильное приложение | v3.0 |
| 5 | Шортовые стратегии | Не планируется |

---

## Критерии готовности MVP

- [x] Бот устанавливается за ≤ 30 минут
- [x] Бот запускается одной командой
- [x] Бот торгует ETH/USDT в автоматическом режиме
- [x] Все сделки логируются
- [x] Уведомления приходят в Телеграм
- [x] Нет цены ликвидации при работе бота
- [x] Бот восстанавливается после разрыва соединения
- [ ] Успешная работа на тестнете минимум 24 часа

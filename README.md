## 💡 Стратегия (DCA)

Бот использует **Dollar-Cost Averaging** (усреднение) — вместо стоп-лосса при падении цены открывает дополнительные позиции, снижая среднюю цену входа:

- **Только LONG** — шорт заблокирован
- **До 5 последовательных входов** при падении цены на 2% между входами
- **Take Profit**: 1% от средней цены входа
- **Плечо**: 4x
- **Размер позиции**: 5% от рабочего депозита
- **Защита от ликвидации** — проверка перед каждым входом

### Результаты бэктестинга
- ✅ Стратегия прибыльна на 2-летних и 4-летних данных ETH/USDT
- ✅ Без ликвидаций на всех тестовых периодах
- ✅ Успешное прохождение через крупные просадки рынка за счёт DCA

## 🚀 Быстрый старт

### 1. Клонировать репозиторий
```bash
git clone <repo-url>
cd "Бот криптотрейдер/src"
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Настроить окружение
```bash
cp .env.example .env
# Отредактировать .env: вставить API-ключи Bybit и Telegram
```

### 4. Настроить стратегию
Параметры в `config.json` (можно оставить дефолтные):
```json
{
  "trading": {
    "symbol": "ETHUSDT",
    "leverage": 4,
    "take_profit_pct": 1.0,
    "max_entries": 5,
    "entry_step_pct": 2.0,
    "position_size_pct": 5.0,
    "working_deposit": 1000
  }
}
```

### 5. Запустить бота
```bash
python main.py start           # Запуск торговли
python main.py status          # Проверка статуса
python main.py stats           # Статистика сделок
python main.py check-config    # Валидация конфигурации
```

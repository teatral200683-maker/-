# 💡 Идея проекта — Бот Криптотрейдер

## Суть проекта

Автоматизированный торговый бот для биржи **Bybit**, который торгует криптовалютой **Ethereum (ETH)** по лонговой стратегии с усреднением позиции и фиксированным тейк-профитом ~1%.

---

## Ключевые принципы работы

### Стратегия
- **Тип**: Лонговый бот (только покупка) — без шортов
- **Монета**: ETH (Ethereum) — топовый актив с низким риском обнуления
- **Цель на сделку**: ~1% прибыли (≈4% с учётом 4x плеча)
- **Максимум позиций**: до 5 одновременных входов (усреднение)
- **Без стоп-лосса**: бот ждёт выхода в плюс, не фиксирует убыток

### Алгоритм входа
1. Цена падает → срабатывают условия → бот открывает **1-ю позицию**
2. Цена продолжает падать → срабатывают условия → **2-я позиция** (средняя цена входа сдвигается вниз)
3. Повторяется до **5 позиций** максимум
4. Цена разворачивается вверх → бот фиксирует **~1% прибыли** от средней цены входа

### Почему «безрисковый» (антихрупкость)
- **Лонговая позиция в пределах депозита** → нет цены ликвидации (в отличие от шорта)
- **ETH не обнулится** — это топ-2 криптовалюта
- **Протестировано на просадках** — бот продолжал зарабатывать даже при падении ETH

## Архитектурная модель (как это устроено)

```
┌─────────────┐     API-ключи      ┌──────────────┐
│   Сайт бота │ ◄──────────────── │  Биржа Bybit  │
```

- Одобрен биржей Bybit

## Важные оговорки и риски

> ⚠️ **Внимание**: несмотря на заявленную «безрисковость», необходимо учитывать:
> - Риск длительной просадки (деньги «заморожены» в позиции)
> - Комиссии за финансирование (funding rate) при длительном удержании
> - Риск изменения условий биржи или API
> - Прошлая доходность не гарантирует будущую
> - Регуляторные риски криптовалютного рынка

# 🐛 Известные проблемы

## Решённые

### 1. `.env` encoding (UTF-16 null characters)
- **Симптом**: `ValueError: embedded null character` при старте
- **Причина**: Файл сохранён в UTF-16 вместо UTF-8
- **Решение**: Пересоздать `.env` в UTF-8 (без BOM)

### 2. Ошибка синхронизации времени (recv_window)
- **Симптом**: `10002 — request timestamp and recv_window exceed server time`
- **Причина**: Часы Windows не синхронизированы с NTP
- **Решение**: Увеличен `recv_window=20000` в `exchange/client.py`

### 3. Ошибка 30208 (maxTradePrice)
- **Симптом**: Market Buy ордер отклоняется с `30208 maxTradePrice`
- **Причина**: Транзиентная ошибка Bybit Testnet
- **Решение**: Retry-логика (3 попытки с задержкой 1с) в `place_order()`

### 4. Отсутствие стоп-лосса
- **Симптом**: Убыточные позиции висели бесконечно
- **Причина**: Не был реализован механизм защиты
- **Решение**: Добавлен `stop_loss_pct=5.0` — автозакрытие при -5%

## Известные ограничения

| Проблема | Влияние | Статус |
|---|---|---|
| WebSocket разрывается каждые ~12 мин | Автореконнект работает | Мониторинг |
| Windows w32time не запущен | Компенсируется recv_window | Некритично |
| Тестнет данные отличаются от mainnet | Результаты приблизительные | Ожидаемо |
